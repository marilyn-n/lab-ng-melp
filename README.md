# Web Application - Melp Lab

Melp is a Web App focussed on showing information about the best restaurants in the city.

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.0.0.

## Installation

This App is in development and needs installation of:

* Node 

### Build Setup

Make sure you are in the repo home directory first.

#### Install NPM Dependencies

`npm install `

#### Run the following command from home directory

``` bash

 ng serve

```
### Run App

Navigate to http://localhost:4200 to see the app running.
